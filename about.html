About This Site

Welcome to the MIAW Host Demo website. This page is part of a sample project used to test Salesforce Messaging for In-App & Web (MIAW) integrations, modal overlays, and cross–window communication using postMessage().

The purpose of this simple about page is to demonstrate how additional static pages can be hosted on GitHub Pages or Netlify alongside your main chat integration page.
